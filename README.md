# Python Data Analysis Application

A Python application for grouped data analysis, descriptive statistics, CSV handling, and histogram visualisation.

## Overview
This project allows users to input continuous numerical data or load it from a CSV file, group the values into equal-width classes, and generate useful statistical summaries.

It was developed to apply practical Python programming skills including:
- functions and modular design
- user input validation
- file handling with CSV
- data processing
- statistical analysis
- data visualisation

## Features
- Enter continuous numerical data manually
- Save data to CSV
- Load existing data from CSV
- Group data into equal-width intervals
- Generate descriptive statistics, including:
  - mean
  - median
  - mode
  - variance
  - standard deviation
  - modal class
- Display a grouped frequency table
- Plot a histogram of the results

## How to Run
### Requirements
- Python 3.x

### Install dependencies
```bash
pip install pandas numpy matplotlib
```

### Run the program
```bash
python dataCollector.py
```
